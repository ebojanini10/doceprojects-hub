08
Color System — Primary

DOCEPROJECTS Purple

HEX #6F1D8E RGB 111 / 29 / 142 CMYK 63 / 92 / 0 / 44

Usage

The primary purple is the core brand identifier. It is used in the logo, headlines, accent elements, buttons, and key UI highlights. It must never be diluted with opacity or mixed with other hues.

This color conveys authority, precision, and creative intelligence. Use it intentionally — never decoratively.
