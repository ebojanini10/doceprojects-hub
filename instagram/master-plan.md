## Notas de produccion

- **Herramienta principal para disenos:** Canva Pro (subir brand kit con colores y fuentes) o capturar directamente desde los HTML adjuntos
- **Herramienta para Reels:** Screen Studio (Mac) para grabaciones de pantalla + CapCut para edicion
- **Scheduling:** Later o Buffer para programar publicaciones
- **Tamano feed posts:** 1080x1080 px
- **Tamano carousels:** 1080x1350 px (4:5)
- **Tamano stories/reels:** 1080x1920 px (9:16)
- **Fuentes:** Bebas Neue (titulares), Inter (cuerpo)
- **Colores:** Purple #6A1B9A, Navy #0F172A, White #FFFFFF, Black #111111
